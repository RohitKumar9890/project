## üìù **Step-by-Step Actions**

### **Action 1: Create GitHub Repository**
1. Go to: https://github.com/new
2. Name: `edueval`
3. Visibility: Private
4. ‚ö†Ô∏è DO NOT check any initialization options
5. Click "Create repository"

### **Action 2: Push Code to GitHub**
```powershell
# Replace YOUR-USERNAME with your GitHub username
git remote add origin https://github.com/YOUR-USERNAME/edueval.git
git branch -M main
git push -u origin main
```

### **Action 3: Deploy to Render**
1. Go to: https://render.com/register
2. Sign up with GitHub
3. New + ‚Üí Web Service
4. Connect repository
5. **Configure:**
   - Name: `edueval-api`
   - Region: Oregon
   - Branch: `main`
   - Root Directory: `server` ‚Üê **IMPORTANT!**
   - Build: `npm install`
   - Start: `npm start`
   - Free tier
6. Add environment variables (copy from above section)
7. Create Web Service
8. Wait 5-10 minutes
9. **Copy your URL!**

### **Action 4: Deploy to Vercel**
1. Go to: https://vercel.com/signup
2. Sign up with GitHub
3. Add New ‚Üí Project
4. Import your repository
5. **Configure:**
   - Framework: Next.js
   - Root Directory: `client` ‚Üê **IMPORTANT!**
6. Add environment variable:
   ```env
   NEXT_PUBLIC_API_URL=https://YOUR-RENDER-URL.onrender.com
   ```
7. Deploy
8. Wait 3-5 minutes
9. **Copy your URL!**

### **Action 5: Update CORS**
1. Render Dashboard ‚Üí Your service ‚Üí Environment
2. Update:
   ```env
   CORS_ORIGIN=https://your-vercel-url.vercel.app
   CLIENT_URL=https://your-vercel-url.vercel.app
   ```
3. Save (auto-redeploys)

### **Action 6: Seed Admin**
1. Render Dashboard ‚Üí Your service ‚Üí Shell
2. Run: `npm run seed:admin`

### **Action 7: Test!**
- Visit your Vercel URL
- Login: `admin@edueval.local` / `Admin@12345`
